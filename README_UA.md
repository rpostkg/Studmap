# Studmap

Веб-додаток vue для навігації по коледжу, що має функції такі як підтримка PWA, панорами та, можливо, досить цікавий спосіб пошуку кімнати в приміщенні.

## Налаштування проекту
Нічого надзвичайного.
```sh
npm install
```

### Компіляція та гаряче перезавантаження для розробки

```sh
npm run dev
```

### Перевірка типів, компіляція та мінімізація для виробництва

```sh
npm run build
```

### Перевірку PWA можливо зробити через preview конфігурацію
```sh
npm run preview
```
